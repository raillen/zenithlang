# FAQ de licenca e marca

> Audience: user, contributor
> Status: current
> Surface: public
> Source of truth: no

## Resumo

Zenith usa licenca dual:

- Apache-2.0;
- MIT.

Voce pode escolher uma delas para uso do codigo.

Marca e logo seguem regras separadas.

## Posso usar o codigo em projeto proprio?

Sim.

Use os termos de uma das licencas:

- `LICENSE-APACHE`;
- `LICENSE-MIT`.

## Posso fazer fork?

Sim.

Mantenha os avisos de licenca exigidos pela licenca escolhida.

## Posso vender servico baseado em Zenith?

Sim, respeitando a licenca escolhida.

Isso nao concede automaticamente permissao ampla de marca.

## Posso usar o nome ou logo Zenith?

Depende.

Leia:

- `TRADEMARK_POLICY.md`

## Posso contribuir?

Sim.

Contribuicoes seguem o modelo:

```text
Apache-2.0 OR MIT
```

Leia:

- `CONTRIBUTING.md`

## Isto e aconselhamento juridico?

Nao.

Este documento e um guia operacional para facilitar leitura.

Para uso juridico formal, leia os arquivos de licenca completos.
