#Documentos públicos

> Documentación del sitio web y uso del idioma.
> Audiencia: usuario
> Superficie: pública

## Objetivo

Enseñe Zenith con baja fricción.

Utilice esta carpeta para:

- primeros pasos;
- instalación;
- tutoriales;
- pequeños ejemplos;
- guías de uso de herramientas;
- material que puede ir en el sitio web.

No utilice esta carpeta para:

- hoja de ruta;
- lista de verificación;
- informe;
- auditoría;
- planes internos;
- funciones aún no implementadas como si fueran actuales.

## Secciones

- `learn/`: recorrido didáctico para aprender el idioma.
-`cookbook/`: Respuestas prácticas a problemas comunes.
-`history-and-manifesto.md`: historia, motivación y filosofía pública del proyecto.
-`guides/`: guías prácticas existentes.
-`tools/`: Herramientas ecosistémicas como ZPM.
-`licensing/`: licencia, marca registrada y contribución.
-`get-started/`: primeros pasos.
-`language/`: guías de enseñanza de idiomas.
-`stdlib/`: guías de uso de stdlib.
-`packages/`: guías oficiales de paquetes.

## Regla para el contenido del usuario

Cada página publicada debe responder rápidamente:

- qué es;
- cuándo utilizar;
- cómo escribir;
- error común;
- límite actual.

Usar`docs/internal/standards/user-doc-template.md`para mantener este estándar.
