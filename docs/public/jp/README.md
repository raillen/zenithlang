# パブリックドキュメント

> Web サイトと言語の使用に関するドキュメント。
> 対象者: ユーザー
> 表面: 公開

＃＃ 客観的

低フリクションのゼニスを教えてください。

このフォルダーを使用して次のことを行います。

- 最初のステップ;
- 設置;
- チュートリアル;
- 小さな例;
- ツールの使用ガイド。
- ウェブサイトに掲載できる素材。

このフォルダーは次の用途には使用しないでください。

- roadmap;
- checklist;
- 報告;
- 監査;
- internal plans;
- 機能はまだ実装されていないため、まるで最新のものであるかのように扱われます。

## セクション

- `learn/`: 言語を学ぶための教訓的な道。
-`cookbook/`: 一般的な問題に対する実践的な回答。
-`history-and-manifesto.md`: プロジェクトの歴史、動機、公共哲学。
-`guides/`: 既存の実践ガイド。
-`tools/`: ZPM などのエコシステム ツール。
-`licensing/`: ライセンス、商標、および貢献。
-`get-started/`：最初のステップ。
-`language/`: 言語教育ガイド。
-`stdlib/`: stdlib の使用ガイド。
-`packages/`: 公式パッケージガイド。

## ユーザーコンテンツのルール

公開されたすべてのページは迅速に応答する必要があります。

- それは何ですか;
- いつ使用するか。
- 書き方;
- 一般的なエラー。
- 電流制限。

使用`docs/internal/standards/user-doc-template.md`この基準を維持するために。
