# Zenith 0.1.x: Genesis

> Status: `Archived`
> Data de referência: `2026-03-02`

`Genesis` e o nome da linha `0.1.x` do Zenith. Ela representa o momento em que a linguagem deixa de ser apenas vontade e passa a existir como sistema funcional.

## O que Genesis significou

- primeira linha realmente utilizavel da linguagem
- transpilation inicial para Lua
- estruturas básicas de variaveis, funções e dados
- fundação emocional e técnica do Zenith

## Valor histórico

Sem Genesis, não existiria Milestone. Eu preservo essa linha como:

- memoria da origem
- ponto de comparação
- registro do comeco desse sonho
