# Zenithマスタータイムライン (v0.3 -> v1.0)

このドキュメントは、Zenithの進化を一つのビジョンにまとめ、Ascensionが公式エンジンとして完成したマイルストーンから始まります。

## 1. 現状: Zenith Ascension (v0.3.0)

Zenithは正式にセルフホスト（自己再帰的コンパイル）のフェーズにあります。

- ステータス: セルフホスト済み、運用中、ドキュメント化済み
- ハイライト: 完全なセルフホスティング、エンジンの最適化、マルチプラットフォーム配信
- 品質: ZTestを介した統合テストスイート
- リファレンス: [Core](core.md) | [Ascension](ascension.md)

## 2. 次のサイクル: 産業的な習熟 (v0.4 -> v0.9)

すでに完成した基盤上での、ツーリング、人間工学、および生産性の拡大。

- フォーカス: Compass、Atlas、Keter、および新しい生産性レイヤー
- 目標: 統合カバレッジ、観測可能性、およびパッケージングの拡大
- リファレンス: [品質ロードマップ](quality.md)

## 3. 未来ビジョン: Zenith v1.0 Odyssey

v1.0への道は、UI、ウェブ、ゲーム、および自動化のための広範なプラットフォームとしてZenithを見据え続けています。

- ステータス: 計画中
- ターゲットフレームワーク: Zenith Meridian、Zenith Borealis、Zenith Intersect、およびZenith Orion
- リファレンス: [Odysseyビジョン](odyssey.md)
