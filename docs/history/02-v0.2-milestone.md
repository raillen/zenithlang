# Marco Histórico: Zenith Core v0.2

**Objetivo**: Registrar o estado da linguagem após a conclusão do Core v0.2.  
**Público-alvo**: Desenvolvedores do núcleo e historiadores do projeto.  
**Contexto**: Versão 0.2 (Abril 2026)

## Conteúdo Principal

O fechamento do **Zenith Core v0.2** marca a transição de um protótipo sintático para um compilador de força industrial. Nesta versão, o Zenith deixou de ser apenas "Lua com outra pele" e passou a ter comportamentos semânticos complexos e independentes.

### As Três Grandes Revoluções do v0.2

1. **O Motor de Match**: Implementação de `match` com destruturação real. O compilador agora consegue "olhar" dentro de listas e structs durante o desvio condicional, gerando código Lua otimizado.
2. **Sistema de Traits (Protocolos)**: O Zenith agora suporta polimorfismo baseado em interfaces. Um `apply` garante que um `struct` cumpra um contrato, fornecendo segurança de tipos em tempo de compilação.
3. **Reatividade Automática**: Introdução de `state`, `computed` e `watch` integrados ao transpilador. O Zenith v0.2 automatiza a gestão de estado, transformando atribuições simples em chamadas de sistema reativas.

## Decisões e Justificativas

- **Decisão**: Múltiplos padrões no mesmo `case` (ex: `case "A", "B":`).
- **Motivo**: Reduzir a repetição de código (DRY) e manter a legibilidade em decisões complexas.
- **Implementação**: Transpilado para cadeias de `or` no Lua para manter a performance.

## Exemplos Comentados

```zt
-- Exemplo de Match com Destruturação (Feature v0.2)
match status
    case 200, 201:
        print("Sucesso!")
    case Error { code, message }:
        print("Erro " + code + ": " + message)
    else:
        print("Desconhecido")
end
```
*No v0.2, o segundo case automaticamente cria as variáveis locais `code` e `message` se `status` for um struct do tipo `Error`.*

## Impactos na Implementação
- **Binder**: Adicionado sistema de `_bind_pattern` para gerenciar escopos de captura.
- **Enum Handling**: Padrões agora reconhecem membros de Enum automaticamente.
- **REPL/Runtime**: A biblioteca `zenith_rt.lua` foi expandida para suportar o motor de watch.

**Status**: Finalizado (v0.2)
