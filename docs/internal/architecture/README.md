# Internal Architecture

> Arquitetura tecnica interna do repositorio.
> Audience: contributor, maintainer
> Surface: internal
> Status: current

## Objetivo

Agrupar mapas tecnicos e pontos de entrada para entender o codigo.

Nem toda documentacao tecnica precisa sair da pasta do subsistema.

Documentos como `compiler/*_MAP.md`, `runtime/c/README.md`, `stdlib/README.md` e `tests/README.md` podem ficar perto do codigo quando isso melhora manutencao.

- Runtime C: `runtime/c/README.md`.
- Stdlib: `stdlib/README.md`.
- Testes: `tests/README.md`.
