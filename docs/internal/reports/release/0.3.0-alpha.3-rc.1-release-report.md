## Residual Risk

The language surface for R3 is broad.

The biggest residual risk is not syntax correctness. It is release confidence:

- baseline drift may hide real runtime/compiler cost;
- Borealis fallback behavior may be environment-sensitive;
- differential validation failure may point to a semantic mismatch.

Until these are resolved, this candidate should stay unpublished.
