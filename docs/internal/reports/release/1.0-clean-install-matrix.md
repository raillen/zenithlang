# Zenith 1.0 Clean Install Matrix

> Audience: maintainer
> Status: Phase 8 readiness artifact
> Date: 2026-04-28

The 1.0 clean install gate covers three platforms.

## Required Matrix

| Platform | Evidence path | Required commands |
|---|---|---|
| Windows x64 | GitHub Actions `windows-latest` | `python build.py`, `zt check zenith.ztproj --all --ci`, `python run_suite.py smoke --no-perf` |
| Linux x64 | GitHub Actions `ubuntu-latest` | `python build.py`, `zt check zenith.ztproj --all --ci`, `python run_suite.py smoke --no-perf` |
| macOS arm64/x64 runner | GitHub Actions `macos-latest` | `python build.py`, `zt check zenith.ztproj --all --ci`, `python run_suite.py smoke --no-perf` |

## Current State

The workflow is defined in `.github/workflows/ci.yml`.

This local Windows session cannot execute Linux or macOS. The GitHub Actions run
after push is the cross-platform clean checkout evidence.

## Release Rule

Before a public 1.0 tag:

- all three matrix jobs must pass;
- failures must be fixed or documented as release blockers;
- a release report must link to the successful workflow run;
- the local workspace must be clean before packaging.
