# R3.M6 Closures v1 Report

- Date: 2026-04-24
- Status: complete with one unrelated suite failure
- Scope: restricted closures, immutable capture, closure ABI, runtime lifetime

## Summary

R3.M6 is implemented.

Zenith now supports explicit closure expressions with the `func ... end` form.

Closures v1 capture outer values by immutable snapshot.

Mutable capture is intentionally blocked in this cut.

## Delivered

- `func(...)` values are represented as managed closure fat pointers.
- Generated Zenith functions receive internal `zt_ctx`.
- Direct calls pass `NULL` for `zt_ctx`.
- Indirect closure calls unpack and pass closure context.
- Captured managed values are retained on closure creation.
- Captured managed values are released through the closure context drop hook.
- Mutating a captured variable emits `closure.mut_capture_unsupported`.
- Dyn trait wrappers were updated to call the new function ABI.
