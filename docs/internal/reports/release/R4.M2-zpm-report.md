# R4.M2-zpm-report - ZPM MVP Report

Date: 2026-04-25
Status: fechado como MVP local
Milestone: R4.M2 - Zenith Package Manager MVP

## Resumo

ZPM esta integrado ao driver local e tambem possui binario standalone.

Entradas aceitas:

- `zt zpm ...`
- `zt pkg ...`
- `zpm.exe ...`

O objetivo deste corte e permitir fluxo local previsivel de pacote:

- criar projeto;
- adicionar dependencia;
- gerar lockfile;
- validar publicacao minima.

## Entregas

- [x] `zpm init`: cria `zenith.ztproj`.
- [x] `zpm add <pkg>`: atualiza dependencias no manifesto.
- [x] `zpm install`: resolve dependencias conhecidas e gera `zenith.lock`.
- [x] Lockfile v1: serializacao em `zenith.lock`.
- [x] `zpm publish`: validacao minima com check/doc check.
- [x] `zt pkg`: alias de leitura mais natural para o usuario.
- [x] `zpm.exe`: binario standalone gerado pelo build local.

## Correcoes deste fechamento

- O build de `zpm.exe` foi restaurado.
- O standalone reutiliza o driver sem duplicar `main`.
- Specs de versao normalizadas pelo parser agora entram corretamente no lockfile.
- Dependencia local como `localpkg@1.2.3` grava `version = "1.2.3"`.
- Buffers de paths do ZPM foram ampliados para reduzir truncamento em caminhos Windows longos.

## Arquivos principais

- `compiler/driver/main.c`
- `compiler/driver/zpm_main.c`
- `compiler/driver/zpm.c`
- `compiler/driver/zpm.h`
- `compiler/project/lockfile.*`
- `language/spec/lockfile-schema.md`

## Validacao

Comandos executados:

```powershell
python build.py
```

```powershell
./zt.exe zpm help
./zt.exe pkg help
./zpm.exe help
```

Smoke manual de pacote:

```powershell
./zpm.exe --project <tmp> init
./zpm.exe --project <tmp> add localpkg@1.2.3
./zpm.exe --project <tmp> list
./zpm.exe --project <tmp> install
```

Lockfile esperado:

```toml
[[package]]
name = "localpkg"
source = "version"
version = "1.2.3"
```

## Limites conhecidos do MVP

- Registry remoto real ainda e trilha futura.
- Preservacao perfeita de comentarios/formatacao do manifesto ainda nao e garantia.
- Resolucao semantica de ranges complexos depende de maturar o registry.
