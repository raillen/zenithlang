# Zenith 0.3.0-alpha.3 Compatibility Report

Status: release compatibility accepted
Date: 2026-04-24

## Compatibility Decision

`0.3.0-alpha.3` is acceptable as the final R3 alpha cut.

## Compatible Behavior

- Existing `public const` behavior is preserved.
- `public var` is namespace-level public state, not a global variable system.
- External code can read `public var` through qualified access.
- External code cannot write another namespace's `public var`.
- Closures v1 capture immutable values.
- Lambdas v1 are expression-only.
- `lazy<int>` is explicit and one-shot.
- No implicit lazy evaluation was introduced.

## Deferred Behavior

These are not compatibility regressions.

They are planned future scope:

- `task`;
- `channel`;
- `Shared<T>`;
- runtime race/cancellation behavior;
- generic lazy runtime;
- FFI callbacks.

Clean install from the final package passed.
