## Validação de Spans e Mensagens

Os property tests validam indiretamente spans e mensagens através de:

1. **ASSERT_HAS_ERROR**: Verifica que o código de diagnóstico correto foi gerado
2. **Fuzzing (R2.M2)**: Exercita caminhos de erro com inputs malformados
3. **Behavior tests**: Validam mensagens de erro em cenários E2E

### Exemplo de Validação

```c
#define ASSERT_HAS_ERROR(result, expected_code, msg) do { \
    tests_run++; \
    int _found = 0; \
    for (size_t _i = 0; _i < (result).diagnostics.count; _i++) { \
        if ((result).diagnostics.items[_i].code == (expected_code)) { \
            _found = 1; \
            break; \
        } \
    } \
    if (_found) { tests_passed++; } \
    else { fprintf(stderr, "FAIL: %s\n", msg); } \
} while(0)
```
