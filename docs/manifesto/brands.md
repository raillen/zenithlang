# Marcas e Ecossistema Zenith

Este documento define as identidades oficiais das ferramentas e frameworks que compõem o ecossistema Zenith. Estes nomes devem ser usados consistentemente em toda a documentação, código e comunicação oficial.

## 🌌 Core & Engine

### **Borealis** (anteriormente Z-Rhythm)
O motor de jogo principal do Zenith. Projetado para unir ergonomia de desenvolvimento com performance industrial e ferramentas visuais integradas.

### **Prism** (anteriormente Z-Core Graphics)
A camada base de renderização e áudio. Atua via FFI para falar diretamente com o hardware e bibliotecas nativas, garantindo o máximo de fidelidade.

### **Asterism** (anteriormente Z-Tree)
O sistema de cena baseado em árvore de nós. É através do Asterism que compomos a lógica, a física e os visuais de uma aplicação ou jogo.

---

## 🖥️ UI & Web

### **Meridian** (anteriormente Z-Aura UI)
Framework declarativo de UI para desktop e web. Focado em bindings reativos e uma estética visual premium "out of the box".

### **Intersect** (anteriormente Z-Fiber)
Framework MVVM especializado para o navegador. Permite rodar Zenith nativamente no browser com estado reativo e integração profunda com o DOM.

### **Sentinel** (anteriormente Z-Pulse)
Sistema de automação de interface. Controla teclado, mouse e janelas de forma programática.

---

## 🌐 Servidores & Dados

### **Orion** (anteriormente Z-Nexus)
Servidor HTTP assíncrono orientado a corrotinas. Ideal para APIs escaláveis e serviços de alta performance.

### **Atlas** (anteriormente Z-Query)
Camada de persistência e ORM tipo-segura. Suporta SQLite, PostgreSQL e garante que os dados do negócio sigam as regras da linguagem.

### **Raz** (anteriormente Zenith Vault)
Cofre de segredos e configurações sensíveis. Gerencia criptografia em repouso e acesso seguro a credenciais.

---

## 🛠️ Ferramentas & Inteligência

### **Keter**
A IDE oficial do Zenith. Um ambiente de desenvolvimento projetado para reduzir a carga cognitiva e aumentar a gratificação imediata.

### **Forge** (anteriormente The Forge)
O editor visual e ambiente de autoria. É onde a engine Borealis ganha vida de forma interativa.

### **Compass** (anteriormente Zenith LSP)
O Language Server oficial. Provê inteligência de código, refatoração e diagnósticos precisos.

### **Astrolabe** (anteriormente Zenith Shell)
Shell interativo para automação e gerenciamento de servidores.

### **Orbit**
O repositório e ecossistema de pacotes. Onde a comunidade compartilha e consome módulos Zenith.

### **Tempo** (anteriormente Z-Bench)
Ferramenta analítica para benchmark, latência e análise de memória.

### **Noema** (anteriormente Z-Brain)
Bindings para inteligência artificial e inferência com tensores/ONNX.
