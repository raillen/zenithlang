# Marcas y Ecosistema Zenith

Este documento define las identidades oficiales de las herramientas y frameworks que integran el ecosistema Zenith. Estos nombres deben usarse de forma consistente en toda la documentación, el código y la comunicación oficial.

## 🌌 Núcleo y Motor

### **Borealis** (anteriormente Z-Rhythm)
El motor de juegos principal de Zenith. Diseñado para unir la ergonomía de desarrollo con el rendimiento industrial y herramientas visuales integradas.

### **Prism** (anteriormente Z-Core Graphics)
La capa base de renderizado y audio. Actúa vía FFI para hablar directamente con el hardware y las librerías nativas, garantizando la máxima fidelidad.

### **Asterism** (anteriormente Z-Tree)
El sistema de escena basado en un árbol de nodos. A través de Asterism es como componemos la lógica, la física y los visuales de una aplicación o juego.

---

## 🖥️ UI y Web

### **Meridian** (anteriormente Z-Aura UI)
Framework declarativo de UI para escritorio y web. Enfocado en bindings reactivos y una estética visual de alta gama ("out of the box").

### **Intersect** (anteriormente Z-Fiber)
Framework MVVM especializado para el navegador. Permite ejecutar Zenith de forma nativa en el browser con estado reactivo e integración profunda con el DOM.

### **Sentinel** (anteriormente Z-Pulse)
Sistema de automatización de interfaz. Controla teclado, ratón y ventanas de forma programada.

---

## 🌐 Servidores y Datos

### **Orion** (anteriormente Z-Nexus)
Servidor HTTP asíncrono orientado a corrutinas. Ideal para APIs escalables y servicios de alto rendimiento.

### **Atlas** (anteriormente Z-Query)
Capa de persistencia y ORM de tipo seguro. Soporta SQLite, PostgreSQL y garantiza que los datos de negocio sigan las reglas del lenguaje.

### **Raz** (anteriormente Zenith Vault)
Bóveda de secretos y gestión de configuraciones sensibles. Gestiona el cifrado en reposo y el acceso seguro a credenciales.

---

## 🛠️ Herramientas e Inteligencia

### **Keter**
La IDE oficial de Zenith. Un entorno de desarrollo diseñado para reducir la carga cognitiva y aumentar la gratificación inmediata.

### **Forge** (anteriormente The Forge)
El editor visual y entorno de autoría. Es donde el motor Borealis cobra vida de forma interactiva.

### **Compass** (anteriormente Zenith LSP)
El Language Server oficial. Proporciona inteligencia de código, refactorización y diagnósticos precisos.

### **Astrolabe** (anteriormente Zenith Shell)
Shell interactivo para la automatización y gestión de servidores.

### **Orbit**
El repositorio de paquetes y ecosistema. Donde la comunidad comparte y consume módulos Zenith.

### **Tempo** (anteriormente Z-Bench)
Herramienta analítica para benchmarks, latencia y análisis de memoria.

### **Noema** (anteriormente Z-Brain)
Bindings para inteligencia artificial e inferencia con tensores/ONNX.
