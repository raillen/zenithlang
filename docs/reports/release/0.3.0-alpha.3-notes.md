# Zenith 0.3.0-alpha.3 Notes

Status: R3 release finalized with local verifiable artifacts
Date: 2026-04-24
Cycle: R3

## Summary

This release finalizes the R3 cycle.

The release artifacts are generated in the repository.

No external GitHub Release upload was performed in this workspace step.

## Main Language Changes

- `public var` for namespace state, without introducing `global`.
- Phase 1 concurrency base with explicit boundary-copy helpers.
- FFI 1.0 Phase 1 with ABI, ownership and shielding rules.
- Minimum `dyn Trait` dispatch subset.
- Typed callables and delegates.
- Closures v1 with immutable capture.
- Expression lambdas and integer HOF helpers.
- Explicit one-shot `lazy<int>`.

## Formal Scope Decisions

- Full concurrency runtime surface moved to `R4.CF1` by Decision 091.
- R3 does not ship `task`, `channel`, `Shared<T>` or runtime race/cancellation behavior.
- R3 ships only the Phase 1 concurrency base.
