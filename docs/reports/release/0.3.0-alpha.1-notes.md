## Quick Install (Windows)

1. Extract `zenith-0.3.0-alpha.1-windows-amd64.zip`.
2. Set `ZENITH_HOME` to the extracted folder.
3. Run:
   - `zt.exe check <project>/zenith.ztproj`
   - `zt.exe build <project>/zenith.ztproj`
   - `zt.exe run <project>/zenith.ztproj`
