# Zenith 0.3.0-alpha.2 Notes

Date: 2026-04-23

## Scope

Patch cut on top of `0.3.0-alpha.1`. No new surface language features. The goal is to close the 10 residual correctness pendings tracked in `reports/pending-language-issues-current.md` (`PLI-01` through `PLI-10`) and promote the formatter idempotence gate to a mandatory `pr_gate` step.

## Highlights

- Formatter is now idempotent on every canonical case: `fmt(fmt(x)) == fmt(x)`.
- Idempotence gate (`tests/formatter/run_formatter_idempotence.py`) is wired into `pr_gate` tooling.
- Emitter now detects and fails fast on mangled-symbol collisions, with new unit coverage.
- Lexer has a structural guard against unbounded string literal growth.
- Parser comment handling no longer leaks tokens across nodes (root cause of the `PLI-10` oscillation).
- Driver no longer frees source text prematurely (`PLI-07`).

## Behavior Changes

- `zt fmt --check` will now fail (exit non-zero) on any input whose first pass is not stable under a second pass. Previously some canonical cases silently oscillated.
- `zt check` / `zt build` will now abort with an explicit diagnostic when two Zenith symbols produce the same mangled C name, instead of emitting undefined-behavior C output.

## Compatibility

- Alpha-phase policy still applies: breaking changes are allowed between pre-releases.
- No intentional breaking change to user-visible syntax in this cut. Files that previously survived `zt fmt` by accident (through the now-fixed comment-stealing paths) may now format differently; the new output is canonical and reproducible.

## Known Limits

- `stdlib/platform/` remains internal placeholder (not public API).
- LSP/Compass are still not alpha-blocking items.
- No new Windows release artifact is published with this note; `0.3.0-alpha.1-windows-amd64.zip` remains the public alpha install. See the release report for the rationale.

## Evidence

- `python build.py` -> green.
- `python run_suite.py smoke` -> `10/10`.
- `python run_suite.py pr_gate` (backend) -> `6/6`, including new `test_emitter_symbol_collision`.
- `python run_suite.py pr_gate` (tooling) -> `4/4`, including formatter golden + formatter idempotence.
- `reports/pending-language-issues-current.md` -> 0 open (`P0`/`P1`/`P2`/`P3`), 10 fixed.
