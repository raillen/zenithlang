# Borealis Compatibility Report - R3.B9

Data: 2026-04-22

## Escopo validado

- package Borealis como lib Zenith
- check global do package
- estrutura de docs e manifest para futuro fluxo ZPM
- runtime desktop com fallback para stub

## Compatibilidade atual

Compativel:

- `zt check` do package
- exemplos documentais e exemplos stub-safe
- fluxo desktop com fallback seguro quando Raylib nao estiver presente

Parcial:

- execucao nativa de partes da API Borealis que dependem de limites ainda abertos do backend C

## Principais limites de compatibilidade

- tipos `struct` qualificados ainda bloqueiam parte do `run-pass`
- algumas combinacoes de `map`/`optional` ainda exigem trabalho adicional no emitter C

## Conclusao

O Borealis esta compativel com a fase atual de package e onboarding.

A compatibilidade de runtime nativo para toda a superficie da lib ainda depende da proxima rodada no backend C.
