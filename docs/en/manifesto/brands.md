# Zenith Brands and Ecosystem

This document defines the official identities of the tools and frameworks that make up the Zenith ecosystem. These names must be used consistently throughout documentation, code, and official communication.

## 🌌 Core & Engine

### **Borealis** (formerly Z-Rhythm)
The main game engine of Zenith. Designed to combine development ergonomics with industrial performance and integrated visual tools.

### **Prism** (formerly Z-Core Graphics)
The base rendering and audio layer. Operates via FFI to speak directly with hardware and native libraries, ensuring maximum fidelity.

### **Asterism** (formerly Z-Tree)
The node-tree based scene system. It is through Asterism that we compose the logic, physics, and visuals of an application or game.

---

## 🖥️ UI & Web

### **Meridian** (formerly Z-Aura UI)
Declarative UI framework for desktop and web. Focused on reactive bindings and a premium visual aesthetic "out of the box".

### **Intersect** (formerly Z-Fiber)
Specialized MVVM framework for the browser. Allows running Zenith natively in the browser with reactive state and deep DOM integration.

### **Sentinel** (formerly Z-Pulse)
Interface automation system. Controls keyboard, mouse, and windows programmatically.

---

## 🌐 Servers & Data

### **Orion** (formerly Z-Nexus)
Coroutine-oriented asynchronous HTTP server. Ideal for scalable APIs and high-performance services.

### **Atlas** (formerly Z-Query)
Type-safe persistence and ORM layer. Supports SQLite, PostgreSQL, and ensures project data follows the language's rules.

### **Raz** (formerly Zenith Vault)
Secret vault and sensitive configuration management. Manages encryption at rest and secure access to credentials.

---

## 🛠️ Tools & Intelligence

### **Keter**
The official Zenith IDE. A development environment designed to reduce cognitive load and increase immediate gratification.

### **Forge** (formerly The Forge)
The visual editor and authoring environment. This is where the Borealis engine comes to life interactively.

### **Compass** (formerly Zenith LSP)
The official Language Server. Provides code intelligence, refactoring, and precise diagnostics.

### **Astrolabe** (formerly Zenith Shell)
Interactive shell for automation and server management.

### **Orbit**
The package repository and ecosystem. Where the community shares and consumes Zenith modules.

### **Tempo** (formerly Z-Bench)
Analytical tool for benchmarking, latency, and memory analysis.

### **Noema** (formerly Z-Brain)
Bindings for artificial intelligence and inference with tensors/ONNX.
