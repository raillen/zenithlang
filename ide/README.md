# Zenith Keter

Zenith Keter is the official Integrated Development Environment (IDE) for the Zenith programming language. Built with Tauri, React, and TypeScript, it provides a fast, lightweight, and extensible coding experience.

## Features
- **Project Management**: Built-in file navigator, command palette, and tab manager.
- **Language Support**: Syntax highlighting, outline panel, and integrated problems/diagnostics for the Zenith language.
- **Built-in Tools**: Integrated terminal and global search.
- **Modern UI**: Responsive layout and theme engine.

## Prerequisites
- [Node.js](https://nodejs.org/) (v18 or newer)
- [Rust](https://www.rust-lang.org/)
- OS-specific dependencies for [Tauri](https://tauri.app/v1/guides/getting-started/prerequisites)

## Setup & Development

```bash
# Install dependencies
npm install

# Start development server and the Tauri application
npm run tauri dev
```

## Build for Production

```bash
npm run tauri build
```
