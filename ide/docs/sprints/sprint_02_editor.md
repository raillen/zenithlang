# Sprint 2: The Core Editor (Monaco)

**Foco**: O coração da IDE - Edição de código estável.

### ✅ Entregáveis:
1. **Monaco Integration**: Editor funcional integrado aos painéis centrais.
2. **Zenith Syntax**: Highlight sintático para arquivos `.zt` via Monarch grammar.
3. **Web Standards**: Suporte nativo para HTML, CSS e Javascript/Typescript.
4. **I/O File System**: Abrir, editar e salvar arquivos diretamente no disco.

### 🛠️ Tarefas Técnicas:
- Integrar `@monaco-editor/react`.
- Registrar `zenith` como linguagem e injetar a gramática desenvolvida.
- Implementar debounce no salvamento automático (opcional).
- Lógica de sincronização: clicar no arquivo no Navigator troca o modelo no Monaco.
