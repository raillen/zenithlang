# Sprint 3: UI Aesthetics & Accessibility

**Foco**: Polimento visual, Temas e Neuro-acessibilidade.

### ✅ Entregáveis:
1. **Theming Engine**: Sistema de troca de temas com suporte às paletas solicitadas (Nord, Dracula, Neon, etc).
2. **Neuro Focus**: Tema **Zenith Neuro** integrado com fontes ajustáveis (OpenDyslexic).
3. **macOS Polish**: Aplicação final de sombras, cantos arredondados e cores do Xcode 16.
4. **Settings Foundation**: Estrutura do modal de configurações para troca de aparência.

### 🛠️ Tarefas Técnicas:
- Implementar gerenciamento de classes CSS no `<html>` baseado no estado do Zustand.
- Desenvolver o componente `ThemeSwitcher`.
- Garantir contraste 7:1 para temas acessíveis e evitar brancos/pretos puros.
- Adicionar suporte a carregamento dinâmico de Web Fonts.
