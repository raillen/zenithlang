# Sprint 1: Layout & Navigation

**Foco**: Estrutura física e exploração de arquivos.

### ✅ Entregáveis:
1. **Layout Flexível**: Implementação de painéis redimensionáveis estilo Xcode usando `react-resizable-panels`.
2. **Project Navigator**: Sidebar funcional que lista os arquivos do projeto Zenith.
3. **Tab Bar**: Navegação por abas com ícones de tipo de arquivo.
4. **Resizer System**: Handles de redimensionamento minimalistas e precisos.

### 🛠️ Tarefas Técnicas:
- Integrar `react-resizable-panels` no componente principal.
- Criar bridge Tauri para listar arquivos do sistema (Recursivo).
- Desenvolver componente `FileIcon` baseado na extensão do arquivo.
- Salvar estado do layout (painéis abertos/fechados) via Zustand Persist.
