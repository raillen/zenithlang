# Sprint 4: Omni-Search & Settings

**Foco**: Busca global, comandos e internacionalização.

### ✅ Entregáveis:
1. **Omni-Search Bar**: Paleta de comandos interativa (Cmd+P) funcional via `cmdk`.
2. **Context Menus**: Menus de clique direito nativos usando Radix UI ContextMenu.
3. **i18n Support**: Interface traduzível (PT-BR, EN-US) via `i18next`.
4. **Settings Modal Full**: Menu de configurações funcional e persistente.

### 🛠️ Tarefas Técnicas:
- Implementar indexador de arquivos no Rust para busca instantânea.
- Criar comandos de UI (Ex: "Change Theme", "Open Settings") acessíveis via Omni-Search.
- Configurar `i18next` e criar arquivos de localização.
- Implementar UI de mapeamento de Keybinds.
