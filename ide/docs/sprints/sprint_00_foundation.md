# Sprint 0: Foundation & Setup

**Foco**: Chão de fábrica - Inicialização técnica e estética básica.

### ✅ Entregáveis:
1. **Ambiente Tauri**: Projeto inicializado com core Rust e template React/Vite.
2. **Design System Base**: Tailwind configurado com as cores do macOS e fontes do sistema.
3. **Window Effects**: Janela configurada com transparência nativa (Mica/Acrylic) e remoção de bordas padrão.
4. **State Management**: Zustand pronto para gerenciar o estado global das janelas.

### 🛠️ Tarefas Técnicas:
- Executar `npx create-tauri-app@latest`.
- Configurar `tauri.conf.json` para janelas frameless e transparentes.
- Instalar dependências iniciais: `tailwindcss`, `lucide-react`, `zustand`, `motion-react`.
- Criar utilitário de classes para simular o "Glassmorphism" do macOS.
