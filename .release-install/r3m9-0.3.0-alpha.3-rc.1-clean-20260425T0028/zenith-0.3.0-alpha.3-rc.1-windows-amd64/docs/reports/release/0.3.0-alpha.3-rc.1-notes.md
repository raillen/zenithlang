# Zenith 0.3.0-alpha.3-rc.1 Notes

Status: release candidate blocked for final publication
Date: 2026-04-24
Cycle: R3

## Summary

This candidate packages the R3 language work in a verifiable artifact.

It is not the final published R3 release yet.

Reason: the release gates are still red.

## Main Language Changes

- `public var` for namespace state, with external read and owner-only write.
- Concurrency boundary-copy helpers and the first concurrency spec cut.
- FFI 1.0 Phase 1, with ABI rules and shielding documented.
- Minimum `dyn Trait` dispatch subset.
- Typed callables and delegates.
- Closures v1 with immutable capture.
- Expression lambdas and first integer HOF helpers.
- Explicit `lazy<int>` one-shot runtime support.
