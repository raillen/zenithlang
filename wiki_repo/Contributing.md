# Contributing

Short flow:

1. Open issue with minimal context.
2. Create focused branch.
3. Add tests for bug fix or feature.
4. Run project gates.
5. Open PR with concise description.

Contribution licensing:

- Apache-2.0 OR MIT

Recommended commit sign-off:

```text
Signed-off-by: Your Name <your-email>
```

- https://github.com/raillen/zenithlang/blob/master/CONTRIBUTING.md
